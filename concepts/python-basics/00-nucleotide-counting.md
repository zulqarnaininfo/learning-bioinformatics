# 🧠 Nucleotide Counting in DNA

## 📌 Idea
DNA sequences are made of four nucleotides (A, T, G, C). The goal is to count how many times each nucleotide appears in a sequence.

---

## 🐍 How Python solves this

We treat DNA as a string and go through it one character at a time.

For each character:
- If it is "A", increase A count
- If it is "T", increase T count
- If it is "G", increase G count
- If it is "C", increase C count

This is basic **frequency counting using loops and conditions**.

---

## 🔬 Biology

DNA is composed of four nucleotide bases:

- A (Adenine)
- T (Thymine)
- G (Guanine)
- C (Cytosine)

The frequency of these bases can give basic information about the composition of a DNA sequence.

## 💻 Code Idea

This uses:

- for loops to iterate through a string
- if/elif conditions to classify characters
- manual frequency counting logic
