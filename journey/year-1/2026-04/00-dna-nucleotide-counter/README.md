# 🧬 DNA Nucleotide Counter

## 📌 Objective
Count the number of each nucleotide (A, T, G, C) in a DNA sequence using Python.

---

## 🧠 Concept
Basic nucleotide counting in a DNA sequence using loops and conditions.

---

## 💻 How It Works
- A DNA string is given as input
- Each character is checked one by one using a loop
- Counters are updated for A, T, G, and C
- Final counts and total length are displayed

---

## ▶️ Output
The script prints:
- Total length of DNA sequence
- Count of A, T, G, C nucleotides
