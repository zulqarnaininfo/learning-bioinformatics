# RNA Nucleotide Counter from File (v1)

This project reads RNA sequences from a text file, cleans invalid characters, and calculates the frequency of each nucleotide (A, U, G, C). It processes multiple RNA sequences line by line and outputs both cleaned sequences and their nucleotide counts.

## What I Learned

- How to read and process files line by line in Python
- How to combine multiple functions for a full analysis pipeline
- How to clean RNA sequences by filtering valid bases (A, U, G, C)
- How to use `.count()` to calculate nucleotide frequency
