# DNA Complement and Reverse Complement from File

This project reads DNA sequences from a text file, cleans invalid characters, and generates both the complement and reverse complement for each sequence. Each line in the file is treated as a separate DNA sequence.

## What I Learned

- How to reuse functions across different problems
- How to generate complement and reverse complement of DNA
- Importance of returning correct values from functions
- Debugging logical errors (returning wrong variable)  
Mistake: using `return base` instead of `return clean`
- Using slicing (`[::-1]`) for reversing sequences
- Applying file I/O with biological sequence operations
