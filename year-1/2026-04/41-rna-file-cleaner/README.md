# RNA Cleaning from File

This project reads RNA sequences from a text file and removes invalid characters, keeping only valid RNA nucleotides (A, U, G, C). It processes multiple sequences line by line and prints both original and cleaned versions.

## What I Learned

- How to read files line by line using a `for` loop
- How to clean biological sequences using conditional logic
- Importance of `.strip()` to remove hidden characters
- Using `.upper()` to standardize biological data
- How to filter valid RNA bases (A, U, G, C)
