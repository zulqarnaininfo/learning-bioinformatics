# DNA Nucleotide Counter V2

This script is a function-based DNA analysis tool that counts the number of each nucleotide in a DNA sequence. The program first cleans the input DNA sequence and then counts the occurrences of A, T, G, and C using a separate function.

## What I Learned

- How to create multiple functions for different tasks  
- How to reuse a DNA cleaning function  
- How to count specific characters using built-in functions  
- How to return multiple values from a function  
- How to pass processed data between functions  
- How to structure a simple bioinformatics pipeline using functions
