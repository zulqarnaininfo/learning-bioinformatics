# DNA Sequence Validator

Python script that checks whether a DNA sequence is valid or contains invalid characters. It also cleans the sequence by removing invalid characters and reports how many were removed.

## What I Learned

- How to validate DNA sequences  
- How to detect invalid characters in a string  
- How to clean biological data  
- How to use counters to track changes in data  
- How to separate valid and invalid inputs logically
