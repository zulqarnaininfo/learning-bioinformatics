# DNA Palindrome Checker

This is a simple python script that checks whether a DNA sequence is a palindrome. 
In biology, a DNA palindrome means that the sequence is equal to its reverse complement.

## What I Learned

- What a DNA palindrome is in biology  
- How to generate a complement DNA sequence  
- How to reverse a sequence using slicing  
- How to compare two sequences  
- How to clean input DNA sequences
