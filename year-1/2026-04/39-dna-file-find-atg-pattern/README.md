# DNA Pattern Finder (ATG) from File

This project reads DNA sequences from a text file, cleans invalid characters, and finds all positions of the "ATG" pattern in each sequence. Each sequence is processed line-by-line, and all occurrences of the start codon "ATG" are identified.

## What I Learned

- How to search for patterns in sequences using loops
- Difference between `.find()` (single result) and manual pattern scanning (multiple results)
- How slicing (`dna[i:i+3]`) works for pattern matching
- Importance of using lists to store multiple results
- Avoiding unnecessary loops and incorrect data structures (like sets)
