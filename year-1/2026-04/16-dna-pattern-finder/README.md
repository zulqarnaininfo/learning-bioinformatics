# DNA Pattern Finder (Multiple Positions)

This is a beginner-level bioinformatics tool that searches for a specific DNA pattern inside a DNA sequence and returns all positions where the pattern occurs.
It simulates a basic form of biological sequence searching used in genomics.

## What I Learned

- How to search for patterns in a DNA sequence  
- How slicing (`dna[i:i+n]`) can be used for comparison  
- How to use loops with controlled range limits  
- How to store multiple results using lists  
- Why we use `len(sequence) - pattern_length + 1` to avoid index errors  
- How to implement basic pattern matching manually (instead of `.find()`)

## Learn Goal

- Understand more about `len(sequence) - pattern_length + 1` and similar stuffs
