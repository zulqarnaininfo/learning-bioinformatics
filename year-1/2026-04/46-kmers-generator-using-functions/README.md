# K-mers Generator (Functions Version)

This project is a simple bioinformatics tool built in Python that generates k-mers from a DNA sequence using functions. The sequence is cleaned to remove invalid characters and all possible k-mers are generated using a function

## What I Learned

-How to convert logic into a function-based structure
- How to use string slicing for biological sequence analysis
- How to build reusable bioinformatics python functions tools step-by-step
