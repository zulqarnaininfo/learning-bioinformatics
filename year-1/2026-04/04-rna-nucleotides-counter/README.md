# RNA Nucleotide Counter

Python script that convert a DNA sequence into RNA and then count the number of each nucleotides in the RNA sequence. This script is combination of transcription and sequence analysis.

### What I Learned

- Using loops and conditionals in python
- Using simplified and smaller loop in transcription
- Counting characters in a string
- Working with RNA sequence 
- Combining multiple steps in one program
