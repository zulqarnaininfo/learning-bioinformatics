# K-mer Generator

This project is a simple bioinformatics tool written in Python that generates k-mers from a DNA sequence.

A k-mer is a substring of length k taken from a biological sequence.  
```
DNA = AATTG  
k = 2  
k-mers = ['AA', 'AT', 'TT', 'TG']
```

## What I Learned

- How to use string slicing in Python
- How to apply loops with correct range limits
- How to store results using lists
- Basic concept of k-mers in bioinformatics
- How sliding window logic works in sequences
