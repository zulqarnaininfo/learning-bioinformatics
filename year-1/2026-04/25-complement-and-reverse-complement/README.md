# DNA Complement and Reverse Complement (Functions)

This script is a function-based DNA sequence processing tool. It takes a DNA sequence as input, cleans it, generates its complement, and then produces the reverse complement using separate reusable functions.

## What I Learned

- How to divide a program into multiple functions  
- Importance of writing reusable and independent functions  
- How to pass data between functions  
- How to build a step-by-step processing pipeline  
- How DNA complement and reverse complement are computed  
- How modular code improves readability and structure
