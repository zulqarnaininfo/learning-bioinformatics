# DNA GC Content from File

This project reads DNA sequences from a text file, cleans invalid characters, and calculates the GC content percentage for each sequence. Each line in the file is treated as a separate DNA sequence, simulating real-world biological data processing.

## What I Learned

- How to read files line-by-line using `with open()`
- Importance of cleaning data before analysis
- How to calculate GC content using functions
- Understanding and using `.strip()` to remove newline characters
- Difference between raw file data and processed data
- How to combine multiple functions (cleaning + analysis)
- Improved debugging and code structure from previous mistakes
