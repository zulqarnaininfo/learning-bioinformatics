# K-mers from a DNA File

This project is a Python-based bioinformatics tool that generates k-mers from DNA sequences stored in a file.

## What I Learned

- How to read data from a file using `open()` and loop through lines
- Importance of `strip()` and `upper()` for cleaning file input
- How to apply previously written functions to multiple inputs
- How to process multiple biological sequences instead of single input
