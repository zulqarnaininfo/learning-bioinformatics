# RNA Codon Frequency Analyzer from File (v1)

This project reads RNA sequences from a text file, cleans invalid characters, and calculates codon frequency (triplet analysis) for each sequence. It processes multiple RNA sequences line by line and converts them into codons to analyze how often each codon appears.

## What I Learned

- How to read biological data from files line by line
- How to clean RNA sequences by removing invalid characters
- How codons are formed using triplets
- How to use dictionaries to store and count frequency data
