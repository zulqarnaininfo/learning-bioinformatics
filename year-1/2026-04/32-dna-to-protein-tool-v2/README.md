# DNA to Protein Tool v2

This project is a Python-based bioinformatics tool that converts a DNA sequence into a protein sequence. The program follows the central dogma of molecular biology: (DNA → RNA → Protein) . It first cleans the DNA sequence, then transcribes it into RNA, and finally translates it into a protein sequence using the genetic codon table. The tool uses a menu-driven system to allow users to perform each step interactively.

## Features

- DNA sequence cleaning (removes invalid characters)  
- Transcription (DNA → RNA conversion)  
- Translation (RNA → Protein conversion)  
- Start codon (AUG) detection  
- Stop codon handling (UAA, UAG, UGA)  
- Dictionary-based codon mapping  
- Function-based modular design  
- Interactive menu system using `while True`
