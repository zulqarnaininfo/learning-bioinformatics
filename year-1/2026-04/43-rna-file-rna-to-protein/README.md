# RNA to Protein Translator from File

This project reads RNA sequences from a text file, cleans invalid characters, and translates RNA into a protein sequence using a codon dictionary. It processes multiple sequences line by line and outputs cleaned RNA along with their corresponding protein sequences.

## What I Learned

- How to process biological sequence data from files line by line
- How to clean RNA sequences before analysis
- How codon dictionaries map RNA triplets to amino acids
- How to translate RNA into protein using reading frames
- Importance of start codon (AUG) in translation
- How stop codons terminate protein synthesis
- How to combine multiple functions into a biological pipeline (clean → translate → output)
