# DNA Analyzer (String Functions)

Python script that analyzes a DNA sequence using string functions. It counts nucleotides, converts DNA to RNA, and finds the position of the start codon "ATG".

## What I Learned

- Using Python string functions like `.count()`, `.replace()`, and `.find()`  
- Simplifying code compared to loops and conditionals  
- Handling cases where a pattern may not exist  
- Combining multiple operations in one script  

## Learning Update

In earlier scripts, I used loops and conditionals for similar tasks. In this project, I used built-in string functions to make the code shorter, cleaner, and more efficient.
