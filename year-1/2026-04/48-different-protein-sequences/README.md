# Different Protein Sequences

Python script that converts RNA sequences into protein sequences. It also allows viewing the protein in different formats such as full amino acid names, 3-letter codes, and 1-letter codes.

## What I Learned

- How RNA sequences are structured and cleaned before analysis  
- Codon-based translation of RNA into amino acids  
- How to use Python dictionaries for biological mapping (codon → amino acid)   
- Building a simple menu-based program for user interaction
