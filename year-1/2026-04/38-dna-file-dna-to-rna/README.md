# DNA to RNA Transcription from File

This project reads DNA sequences from a text file, cleans invalid characters, and converts the cleaned DNA into RNA sequences. Each DNA sequence is processed line-by-line, making it possible to handle multiple sequences from a file efficiently.

## What I Learned

- How transcription works (DNA → RNA by replacing T with U)
- Reusing cleaning functions before applying biological operations
- Using `.replace()` for sequence transformation
- Applying file I/O to process multiple sequences
