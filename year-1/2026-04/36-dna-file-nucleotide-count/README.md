# DNA Nucleotide Counter from File

This project reads DNA sequences from a text file, cleans invalid characters, and counts the number of each nucleotide (A, T, G, C) in each sequence. Each line in the file is treated as a separate DNA sequence, allowing processing of multiple sequences at once.

## What I Learned
- How to read and process files line-by-line using `with open()`
- Importance of cleaning data before analysis
- How to count elements in a sequence using `.count()`
- How to return and use dictionaries in Python
- Understanding structured output (key-value pairs)
- Reusing functions across different programs
