# Codon List Generator

This project is a beginner-level Python bioinformatics script that splits a DNA sequence into codons (groups of 3 bases). It helps in understanding sequence indexing, slicing, and basic biological data processing.

## What I Learned

- Learned about lists and how they store multiple values in one variable  
- How to use `range()` for stepping through sequences  
- How indexing works in strings  
- How slicing (`i:i+3`) extracts parts of a sequence  
- How to use `.append()` to store results in a list  
- Basic DNA sequence splitting into codons

## Learn Goal

Understand list-based processing and improve my understanding of slicing (`i:i+3`) and how sequence splitting works in practice.
