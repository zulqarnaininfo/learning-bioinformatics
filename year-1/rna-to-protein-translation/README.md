# RNA to Protein Translator (Basic)

This is a beginner-level Python bioinformatics script that simulates the biological process of translation. It converts an RNA sequence into a chain of amino acids (protein) using codon mapping.

## What I Learned
- Learned about the biological process of transcription and translation  
- How RNA is read in triplets called codons  
- How each codon represents a specific amino acid  
- How to use loops with `range()` for sequence processing  
- How slicing (`i:i+3`) is used to extract codons  
- How to build strings step-by-step using `+=`
