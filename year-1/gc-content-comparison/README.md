# GC Content Comparison

Python script that calculates and compares the GC content of two DNA sequences. GC content represents the percentage of Guanine (G) and Cytosine (C) bases in a DNA sequence, which is an important property in molecular biology.

## What I Learned

- How to calculate GC content of a DNA sequence  
- How to count specific nucleotides in a sequence  
- How to use percentages in biological calculations  
- How to compare values between two sequences  
- How to handle basic edge cases like empty sequences
