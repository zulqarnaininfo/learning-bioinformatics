# DNA Mutation Detector

Python script that compares two DNA sequences and detects mutations between them. A mutation is identified when two sequences have different nucleotides at the same position.

## What I Learned

- How to compare two DNA sequences position by position  
- How to detect mutations using index-based logic  
- How to clean input sequences before processing  
- How to handle unequal sequence lengths  
- How to use a flag variable to track conditions

## Learn Goal

- Learn more about index-based logics and syntax
- Learn how to use flag variable and its other uses
