# GC Content Calculator V2

This script is a function-based DNA analysis tool that calculates the GC content of a DNA sequence. The program first cleans the DNA sequence using a function and then calculates the percentage of G and C nucleotides using another function.

## What I Learned

- Importance of return statement in functions  
- Functions return None if no return is used  
- How to correctly use function parameters inside functions  
- How to calculate GC content using nucleotide counting  
- How to combine multiple functions into a pipeline  
- Debugging errors related to NoneType and variable scope
