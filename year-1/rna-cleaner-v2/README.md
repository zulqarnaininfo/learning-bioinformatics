# RNA Cleaner V2

This script is a function-based RNA cleaning tool. It takes an RNA sequence as input, removes invalid characters, and returns both the cleaned RNA sequence and the number of invalid characters removed.

## What I Learned

- How to create reusable functions for sequence cleaning  
- How RNA differs from DNA (U instead of T)  
- How to validate RNA sequences using conditions  
- How to track invalid characters during processing  
- How to return multiple values from a function  
- How to unpack function outputs into variables
