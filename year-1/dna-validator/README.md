# DNA Validator

This Python script checks whether a DNA sequence is valid by ensuring it contains only valid nucleotides (A, T, G, C).

## What I Learned
- Using `.upper()` to standardize input  
- Using `not in` to check for invalid characters  
- Using `break` to stop a loop early  
- Using a flag variable (`is_valid`) to track conditions  

## Learning Update
In my first attempt, I tried to validate the sequence directly inside the loop. Later, I learned that using a flag variable makes the logic clearer and more accurate when checking conditions.
