# Codon Frequency Analyzer

Python script that analyzes an RNA sequence and counts how many times each codon appears. The program splits the RNA sequence into codons and stores their frequency using a dictionary.

## What I Learned

- How to split RNA sequences into codons using slicing  
- How to iterate through a sequence in steps of 3  
- How dictionaries store and update key-value pairs dynamically  
- How to use codon counting for biological sequence analysis  
- How real bioinformatics tools analyze sequence frequency patterns
