# RNA Nucleotide Counter V2

This script is a function-based RNA analysis tool that counts the number of each nucleotide in an RNA sequence. The program first cleans the RNA sequence using a function and then counts the occurrences of A, U, G, and C using another function.

## What I Learned

- How to reuse function-based structure for similar problems  
- Difference between DNA and RNA (T vs U)  
- How to count nucleotides using built-in functions  
- How to return multiple values from a function  
- How to pass cleaned data between functions  
- How to build consistent bioinformatics tools using functions
