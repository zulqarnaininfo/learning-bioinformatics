# DNA Cleaner V2

This python script is an improved version of a DNA cleaning tool built using functions in Python. The program takes a DNA sequence as input, removes invalid characters, and returns both the cleaned DNA sequence and the number of invalid characters removed. This shows how functions can be used to make reusable and efficient bioinformatics tools.

## What I Learned

- How to create user-defined functions using def  
- How to pass input using parameters  
- Difference between parameters and arguments  
- How to return values from a function  
- How to return multiple values from a function  
- How to unpack multiple returned values into variables  
- How to make reusable functions for bioinformatics tasks  
- How to track additional information (invalid characters) during processing
