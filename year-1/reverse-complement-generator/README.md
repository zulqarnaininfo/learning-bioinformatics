# Reverse Complement Generator

Python script that generates the reverse complement of a DNA sequence. The reverse complement is formed by reversing the sequence and replacing each base with its complementary base.

## What I Learned

- How to reverse a sequence using Python (`reversed()`)  
- How to apply base-pairing rules (A-T, G-C)  
- How to combine reverse and complement operations in one step  
- How to build efficient sequence transformation logic  
- How DNA reverse complement is used in real bioinformatics analysis
